## How It Works

```
Scanner → Cluster → Alpha → Execution → Evaluation → Mutation
   │          │         │         │            │           │
   │    Group related   │    Trigger     Score agents  Rewrite
   │    wallets into    │    alerts &    and apply     prompts
Track   entities     Detect   simulated   Darwinian     via LLM
wallet               profit   trades      selection
activity             patterns
```

### The Evolution Loop

Every 5 minutes, HydraNet evaluates all agents:

1. **Score** — accuracy × profitability × success rate × latency
2. **Kill** — remove agents below threshold
3. **Replicate** — clone top performers
4. **Mutate** — LLM rewrites agent prompts to explore new strategies
5. **Spawn** — meta-reasoning generates entirely new agent types

This is **Darwinian AI** — only the best strategies survive.

## Self-Evolving AI Agents

HydraNet doesn't just run agents — it **breeds** them.

The Agent Generator uses meta-reasoning to identify system gaps:
- *"What capability is missing?"*
- *"What agent would improve system ROI?"*

It then generates new agent definitions, validates them via LLM, and deploys — all without human intervention.

Traditional AI tools require humans to write prompts, tune parameters, and manually orchestrate agents. HydraNet does all of this autonomously.
