## The Arena

The web UI at `http://localhost:8000` shows:

- **Live battles** — watch 3 AIs answer in real-time
- **Full critique phase** — see each AI tear apart the others
- **AI Judge scores** — 0-100 on correctness, completeness, clarity, creativity
- **Human voting** — cast your vote, see if you agree with AI
- **"AI was wrong" alerts** — when humans and AI disagree
- **Share buttons** — one click to X, Reddit, Discord
- **Viral result cards** — battle results formatted for sharing
- **ELO leaderboard** — running rankings across all battles
- **Start your own battle** — type any prompt, watch AIs fight
