## How It Works

```
  YOU: "Write a rate limiter"
   │
   ├──→ GPT-4o writes solution A
   ├──→ Claude writes solution B
   ├──→ Gemini writes solution C
   │
   ├──→ GPT attacks Claude: "Not thread-safe"
   ├──→ Claude attacks Gemini: "Wastes memory"
   ├──→ Gemini attacks GPT: "Missing edge case"
   │
   ├──→ AI Judge: Claude wins (88/100)
   ├──→ Humans vote: Claude 48%, GPT 34%, Gemini 18%
   │
   └──→ EVOLVED ANSWER: thread-safe + memory efficient
        + handles all edges = better than all 3
```

### The 4 Phases

| Phase | What Happens | Why It Matters |
|-------|-------------|----------------|
| **Generate** | 3 AIs answer independently | Diverse approaches |
| **Critique** | Each AI attacks the other two | Blind spots exposed |
| **Judge + Vote** | AI scores + humans vote | Two perspectives, disagreements go viral |
| **Evolve** | Best elements combined | Final answer > any individual |

## Quick Start

```bash
git clone https://github.com/q15432123/HydraNet.git
cd HydraNet
pip install -r requirements.txt

# Watch the demo (no API key needed)
python battle/demo_cli.py

# Start the arena (needs OPENAI_API_KEY)
cp .env.example .env
python main.py
# Open http://localhost:8000
```

### Start a battle from Python:

```python
from battle.arena import BattleArena, Combatant

arena = BattleArena()
arena.add_combatant(Combatant("GPT-4o", "gpt-4o", "openai"))
arena.add_combatant(Combatant("Claude", "claude-sonnet", "anthropic"))
arena.add_combatant(Combatant("Gemini", "gemini-pro", "google"))

result = await arena.battle("Implement an LRU cache with O(1) operations")
print(f"Winner: {result.winner}")
print(f"Evolved: {result.evolved_answer}")
```

### Start a battle from API:

```bash
curl -X POST http://localhost:8000/battle \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Write a rate limiter", "evolve": true}'
```
